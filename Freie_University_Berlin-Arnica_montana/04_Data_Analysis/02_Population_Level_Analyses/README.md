# Population Level Analyses
Based on the genotype table, a number of population-level analyses of genetic diversity and genetic distinctiveness can be made. Scripts included here are typically for R, with further processing of the output e.g. in QGIS. Additional analyses and plots were made directly in Excel.

## SSRseq / SSR length data
Includes a script for the comparison of SSRseq & SSR length alleles, a script for the calculation of Private Alleles, rFIS/2, pairwise FST and FST-based PCoA coordinates, and the data analysis script created by Elisabeth HERTEL as part of her BSc thesis.
