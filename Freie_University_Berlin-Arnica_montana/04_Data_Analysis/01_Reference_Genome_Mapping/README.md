# Reference Genome Mapping 
To find out the position of each studied locus, the sequences of the alleles obtained for a clonal sibling of the Arnica reference genome specimen were mapped onto the species' assembled reference genome. 

## SSRseq
The actual mapping was performed in Geneious, using its Bowtie 2 plugin. Included here is an R script to make a draft visualisation (later edited in Inkscape) of the loci on an ideogram-like chromosome plot.
