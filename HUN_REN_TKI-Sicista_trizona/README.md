# A High-Quality Reference Genome to bring one of the Rarest European Mammal Back from the Brink of Extinction
 This repository includes all the scripts used to generate the chromosome-level genome assembly for the BGE study 'A High-Quality Reference Genome to bring one of the Rarest European Mammal Back from the Brink of Extinction'.
