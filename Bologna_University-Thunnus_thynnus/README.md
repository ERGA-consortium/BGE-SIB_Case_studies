# Past and future anthropogenic impacts on Atlantic bluefi n tuna through the eyes of demographic and adaptation analyses on range-wide whole genomes
This repository includes all the scripts used to estimate the impact of recent population decline on the Thunnus thynnus functional genomic variation.
